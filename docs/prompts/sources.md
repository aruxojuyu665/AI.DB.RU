# Исследование AI/DS/ML источников информации

## Глобальная цель проекта
Создание единственного и цельного агрегатора всевозможных AI/DS/ML инструментов, технологий и источников информации

## Твоя задача 
Найти МАКСИМАЛЬНОЕ КОЛИЧЕСТВО РАЗНЫХ источников информации об AI/DS/ML инструментах/технологиях

## ВАЖНО
Необходимо мыслить согласно первым принципам Илона Маска

## Подсказки для поиска

### АТОМИЗАЦИЯ
- Документация (official docs)
- Научные источники (papers, preprints)
- Социальные сети & обсуждения
- Код-хосты (GitHub, Gitee)
- Видео-платформы
- Форумы & сообщества
- Медиа & блоги
- Компании & организации
- Каталоги & агрегаторы

### СИСТЕМАТИЧЕСКИЙ ОХВАТ
- Официальные каналы (docs, GitHub)
- Неофициальные дискуссии (issues, discussions)
- Обзоры пользователей (Reddit, HN comments)
- Интеграции в других инструментах (упоминания)
- Цепочки ссылок (из одного источника в другой)### ЯЗЫКОВОЕ РАЗНООБРАЗИЕ
- Англоязычные источники: GitHub, HF, Medium, Субстек
- Китайские источники: Gitee, Zhihu, CSDN, WeChat, Bilibili
- Мультиязычные: arXiv, Papers with Code

## Ожидаемый Результат
- Минимум 200 источников
- Каталог в Markdown
- Таблица в CSV
